# Shivani'z Beauty Parlour

Premium beauty parlour website — bridal makeup, advanced hair treatments & skincare in Raikot, Punjab. Established 2014.

## Tech Stack

- **React 19** with Create React App + CRACO
- **Tailwind CSS 3** for styling
- **Framer Motion** for animations
- **Lenis** for smooth scrolling

## Getting Started

```bash
cd frontend
npm install
npm start
```

Opens at [http://localhost:3000](http://localhost:3000).

## Deployment

Deployed on **Vercel** with the `frontend` directory set as the root.
